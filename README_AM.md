## ­Ъџђ рЅБрѕЁрѕфрІФрЅх

### рІІріЊ рЅ░рїЇрЅБрѕФрЅх

-   **рЇѕрїБріЋ QR рѕЏрѕўріЋрїерЅх** - ріерѕЏріЋріЏрІЇрѕЮ рїйрѕЉрЇЇ рЅарѕ░ріеріЋрІХрЅй рІЇрѕхрїЦ ріфрІЇ ріарѕГ рі«рІХрЅйріЋ рІГрЇЇрїарѕЕ
-   **рІерЅЁрІхрѕў-рЅЃрѕЇ рІхрїІрЇЇ** - рІерЅ░рѕѕрІФрІЕ рЅЁрІхрѕў-рЅЃрѕјрЅй рІФріГрѕЅ (https://, tel:, mailto:, рІѕрІўрЅ░.)
-   **рІерѕЁрЅхрѕўрЅх рѕхрѕГрІЊрЅх** - рЅарЅ░рѕѕрІФрІЕ рѕўрїЇрѕѕрїФрІјрЅй рІерѕџрїІрѕЕ рїѕрїЙрЅй рІГрЇЇрїарѕЕ
-   **рЅБрѕѕрЅЦрІЎ рЅІріЋрЅІ рІхрїІрЇЇ** - рЅа6 рЅІріЋрЅІрІјрЅй рІГрїѕріЏрѕЇ
-   **PWA рІЮрїЇрїЂ** - ріЦріЋрІ░ рѕърЅБрІГрѕЇ рѕўрЅ░рїЇрЅарѕфрІФ рІГрїФріЉ
-   **SEO рЅ░рѕўрЅ╗рЅйрЅХ** - рѕѕрЇЇрѕѕрїІ рѕърЅ░рѕ«рЅй рЅ│рІГріљрЅх рЅ░рѕ░рѕГрЅирѕЇ

### рІерЅ░рїарЅЃрѕџ рѕЇрѕЮрІх

-   **рЅаріаріЋрІх рїарЅЁрЅ│ рѕЏрѕхрїѕрЅБрЅх** - ріерЅЁріЋрїЦрЅЦ рѕ░рѕїрІ│ рЅарЇЇрїЦріљрЅх рІФрѕхрїѕрЅА
-   **рЅаріЦрІЇріљрЅ░ріЏ рїірІю рѕЏрѕўріЋрїерЅх** - ріфрІЇ ріарѕГ рі«рІХрЅй рЅарѕџрЅ░рІГрІЎрЅарЅх рїірІю рІГрІўрѕўріЊрѕЅ
-   **рѕѕрѕърЅБрІГрѕЇ рЅ░рѕўрЅ╗рЅйрЅХ** - рѕѕрѕърЅБрІГрѕЇ рѕўрѕ│рѕфрІФрІјрЅй рЇЇрї╣рѕЮ
-   **рїерѕѕрѕЏ/рЅЦрѕГрѕЃріЋ рїГрЅЦрїЦ** - ріарІЇрЅХрѕЏрЅ▓ріГ рІерїГрЅЦрїЦ рѕўрЅђрІФрІерѕГ
-   **рѕЏрїІрѕФрЅх ріЦріЊ рѕЏрІЇрѕерІх** - рЅарѕГріФрЅ│ рІерѕЏрїІрѕФрЅх ріарѕЏрѕФрї«рЅй

## ­Ъњг рѕ░рІјрЅй рѕхрѕѕ QR рі«рІХрЅй рѕЮріЋ рІГрѕІрѕЅ

> "QR рі«рІХрЅй рІерѕЏрѕГрігрЅ▓ріЋрїЇ рІўрѕўрЅ╗рІјрЅй рІІріЊ ріаріФрѕЇ рѕєріљрІІрѕЇрЇБ рЅЦрѕФріЋрІХрЅй ріерѕўрѕхрѕўрѕГ рІЇрїГ ріЦріЊ рІерѕўрѕхрѕўрѕГ рІЊрѕѕрѕърЅйріЋ рЅарЅђрѕІрѕЅ ріЦріЋрІ▓рІФрїѕріЊріЎ рІФрѕхрЅйрѕІрѕЅрЇб"

> "рІЏрѕг QR рі«рІХрЅй рЅарѕЂрѕЅрѕЮ рЅдрЅ│ рІГрїарЅђрѕЏрѕЅрЇА ріерѕЮрїЇрЅЦ рЅцрЅХрЅй рѕюріЉрІјрЅй ріЦрѕхріе рІЮрїЇрїЁрЅХрЅй рЅхрігрЅХрЅйрЇБ рІѕрІ░ рѕўрѕерїЃ ріЦріЊ ріарїѕрѕЇрїЇрѕјрЅХрЅй рѕўрІхрѕерѕхріЋ рЅђрѕІрѕЇ рІФрІ░рѕГрїІрѕЅрЇб"

> Рђћ _рІ▓рїѓрЅ│рѕЇ ріарѕГрЅ▓рѕхрЅх_ рЅа r/art

## ­Ъџђ рЇѕрїБріЋ рѕўрїђрѕўрѕфрІФ

### рідріЋрѕІрІГріЋ ріарїарЅЃрЅђрѕЮ

1. [Paste2QR.com](https://paste2qr.com) рІГрїјрЅЦріЎ
2. рїйрѕЉрЇЇрІјріЋ рІФрѕхрїѕрЅА рІѕрІГрѕЮ рІГрЅ░рІГрІЎ
3. QR рі«рІхрІјріЋ рЅарЇЇрїЦріљрЅх рІФрїЇріЎ
4. ріарѕхрЇѕрѕІрїі ріЦріЋрІ░рѕєріљ рІФрїІрѕЕ рІѕрІГрѕЮ рІФрІЇрѕГрІ▒

### рІеріаріФрЅБрЅб рѕЇрѕЏрЅх

```bash
# рѕфрЇќрІџрЅХрѕфрІЇріЋ ріГрѕјріЋ ріарІхрѕГрїЅ
git clone https://github.com/yourusername/paste2qr.git
cd paste2qr

# рїЦрїѕріЏрІјрЅйріЋ рІГрїФріЉ
npm install

# рІерѕЇрѕЏрЅх рѕ░рѕГрЅерѕГ рІГрїђрѕЮрѕЕ
npm run dev

# рѕѕрѕЮрѕГрЅх рІГрїѕріЋрЅА
npm run build
npm start
```
